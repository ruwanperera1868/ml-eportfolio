About this portfolio

This portfolio documents knowledge and skill development across machine learning foundations, exploratory data analysis, regression, clustering, neural networks, convolutional neural networks, NLP, model evaluation, and Industry 4.0. The work reflects both technical experimentation and critical consideration of legal, social, ethical, and professional issues.

The structure has been kept simple so that GitHub Pages can render it reliably from the repository root without any build pipeline.
